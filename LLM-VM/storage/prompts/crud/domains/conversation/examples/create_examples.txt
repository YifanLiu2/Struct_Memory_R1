====================
Examples
====================

### Add observation to existing session
User: "Add an observation that Caroline learned pottery in session 4"
Context: - Speaker Caroline > Session 4 children: [0] Observation: "Caroline received a necklace...", [1] Observation: "Caroline treasures a hand-painted bowl..."
Output:
```json
{
  "parent_path": "Speaker Caroline > Session 4",
  "position": -1,
  "node_type": "Observation",
  "fields": {
    "text": "Caroline learned pottery and enjoyed the creative process.",
    "source_dia_id": "N/A"
  },
  "reasoning": "Position -1 appends after existing observations in session 4. source_dia_id set to N/A since this is a manually added observation."
}
```
