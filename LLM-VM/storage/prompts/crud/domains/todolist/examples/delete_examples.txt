[2] Backend > Create API endpoints (Task, status: In Progress, score: 0.72)
[3] Frontend > Design mockups (Task, status: Done, score: 0.65)

```json
{
  "nodes_to_delete": [
    {"id": "1", "path": "Backend > Setup database", "reasoning": "Done task in Backend project"}

  "reasoning": "Selected done task in Backend. Excluded in-progress task and Frontend task."
}
```
