\begin{abstract}
Memory-augmented Large Language Models (LLMs) maintain long-horizon memory over conversational history to provide relevant context for future interactions. Conversational information has natural event and topic based structure that a tree can capture better than a flat representation. However, deciding how to traverse relevant tree branches to access memory and where in the hierarchy new facts belong to update memory becomes substantially harder than with flat structure. Reinforcement learning (RL) with Group Relative Policy Optimization (GRPO), which uses group-sampled reward normalization as its training objective, has been shown to improve reasoning ability over complex multi-step trajectories. We propose \textsc{Struct Memory-R1}, a framework that maintains tree-structured memory from raw conversation and uses RL with GRPO-style question-answering reward to train three components: a \emph{Memory Manager} for tree updates, a \emph{Retrieve Agent} for tree access, and an \emph{Answer Agent} for downstream question answering. On the LoCoMo benchmark, \textsc{Struct Memory-R1} achieves the highest overall F1 and LLM-judge accuracy among baseline methods, with particularly large gains on multi-hop (+8.6 F1) and temporal (+14.1 F1) questions over the flat-memory RL baseline.
\end{abstract}
