\section{Conclusion}
We introduced \textsc{Struct Memory-R1}, a reinforcement-learning framework that learns tree-structured conversational memory from raw interaction history. The framework decomposes the memory-augmented generation problem into three components---a Memory Manager for structural updates, a Retrieve Agent for tree-based search planning, and an Answer Agent for evidence-conditioned response generation---each trained with GRPO from downstream QA reward. On the LoCoMo benchmark, \textsc{Struct Memory-R1} achieves the highest overall F1 (30.4) and LLM-judge accuracy (54.4) among non-oracle methods, with +9.0 F1 on multi-hop and +13.8 F1 on temporal questions over the flat-memory Memory-R1 baseline. These gains confirm that hierarchical memory organization is most valuable when questions require navigating relationships among facts across sessions. The framework extends the outcome-driven training principle of Search-R1 and Memory-R1 from flat memory to tree-structured access and updates, providing a principled path toward long-horizon conversational agents with learned structured memory.
